## About The Project

The Post-Deyr assessment is integeral part of the IPC process, aimed at collecting data on food security, livelihood and nutrition status at the district and livelihood zone levels.


This repository contains data cleaning scripts designed for daily use to monitor the quality of incoming data and provide constructive feedback to enumerators and supervisors.
